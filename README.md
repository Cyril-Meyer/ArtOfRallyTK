# ArtOfRallyTK
Toolkit for [Art Of Rally](https://artofrally.com/)

### Tools

* [Dashboard](dashboard)  
Read and log speed, RPM and engaged gear of running game
* [Leaderboards](leaderboards)  
Scrap leaderboards using funselektorfun.com api
